# projeto-pimg

O projeto tem como objetivo identificar e classificar pizzas em pequenas, médias e grandes a partir de imagens capturadas com câmera fixa (mesmo ângulo e mesma distância). Ao manter a câmera constante e variar apenas o diâmetro real das pizzas, o raio detectado em pixels pela CHT torna-se proporcional ao tamanho físico da pizza, permitindo a classificação correta por tamanho real.
